# Taskflow PRD (v0.1)

## 1) One-liner

Taskflow is an AI-driven daily productivity hub that helps users plan today, follow habits, and stay consistent through a simple daily flow and actionable coaching.

## 2) Goal

Make self-improvement simple and motivating by combining tasks + habits + a daily plan in one place, with AI that turns user behavior into clear next steps.

## 3) Primary Users (MVP focus)

- Students and early-career builders (creators/coders) who want structure
- People who struggle with consistency (procrastination, poor planning habits)

Everyone else is secondary until after MVP.

## 4) Problem

Users fail to stay consistent because:

- Productivity tools are scattered or overwhelming
- Most apps track tasks/habits but don’t guide improvement
- Users don’t have a simple daily system they can repeat

## 5) Solution

Taskflow provides:

- A Today Page that shows the user exactly what to do next
- Task management + habit tracking in one flow
- Progress/streaks for motivation
- One AI coaching feature that gives actionable guidance (not fluff)

## 6) Unique Selling Point (USP)

Most apps track tasks. Taskflow coaches behavior: personalized AI that uses the user’s tasks/habits/history to generate a realistic plan and improvement suggestions.

## 7) MVP Scope (What we will ship first)

MVP is done when a user can:

- Sign up / log in / log out
- Create/edit/complete/delete tasks
- Create a habit + mark done today + see streak/progress
- Use a Today Page that combines tasks + habits
- Use the chosen MVP AI feature (locked): **Plan my day**
  - Generates a simple day plan using tasks + habits + available time

## 8) Non-goals (Not in MVP)

- Teams / collaboration
- Calendar integrations
- Advanced analytics
- Social features
- Multiple AI tools/features at once

## 9) Success Metrics (MVP)

- User reaches a usable Today plan within 60 seconds of signup, after signup
- User completes at least 1 task + 1 habit in first session
- User returns the next day (basic retention signal)
